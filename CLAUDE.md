# 二手車買賣專案

## 客戶
昇昇賣好車 — 二手車商，9年經驗，採購為主。

## 知識庫
所有客戶資料和報價邏輯在 `knowledge/` 目錄下，生成任何內容前先讀取。

## 提案風格
- 品牌色：深墨綠 `#1B4332` + 橘 `#E76F51`
- 輔助色：`--paper: #f6f4f0` `--cream: #eceae4` `--slate: #4a5e54` `--ghost: #9aab9f`
- 字體：DM Serif Display（標題）+ Noto Sans TC（內文）+ JetBrains Mono（標籤）
- 語氣：專業但不冷、直白不推銷、像朋友在聊天
- 提案頁面用 single-file HTML，不依賴外部 CSS/JS

## 車行術語
- 收車 = 買進二手車
- 散客 = 非車行轉介、自己找上門的賣家
- 板件 = 車身鈑金件
- 權威/建價 = 銀行估價參考
- 台88 = 中古車交易平台（8890嚴選 = 品質認證）
- 換購 = 舊車換新車
