# 報價邏輯

## 三大依據
1. **經驗（70%+）**：年份、里程、版本、車況、板件狀態 → 一車一價
2. **銀行建價**：年份+里程+版本 → 對照表查銀行估價，但實際行情可能差很多（某些車款行情僅建價 80%）
3. **拍賣成交紀錄**：中古車拍賣網站查同款近期成交價

## 流程
1. 收到車輛資訊（車款、年份、里程、照片、事故/現況說明）
2. 依三大依據報價
3. 如確認無重大問題，可不看車直接報接近實際收購價
4. 買進後：鑑定 → 檢查車況 → 定賣點 → 定價策略 → 上架

## 委託拍賣
- 設底價上拍賣平台，超過底價即成交
- 需簽委託書，未成交賠違約金 ~NT$10,000-20,000
